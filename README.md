## Zephyria
JavaFX / Kotlin RPG

## Build
```bash
mvn package
```

## Run
```bash
java -jar
```
produced jar or simply double click it.

## Current State
Early development

## Basic Specification

* A game character has 9 attributes and 16 stats.
* 3 damage types
* 5 elements
* 3 experience types
* 3 status types
