## Zephyria (Kotlin RPG) using FXGL

![Screenshot](https://raw.githubusercontent.com/AlmasB/git-server/master/storage/images/zephyria.png)

## Run

You need JDK 11+ to run the game.

```bash
mvn javafx:run
```

## Package

```bash
mvn javafx:jlink
```
